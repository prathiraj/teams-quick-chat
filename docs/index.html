How it works

Click a contact name and Teams opens straight to the 1:1 chat — powered by the Microsoft Teams deep link protocol:

msteams:/l/chat/0/0?users=alice@contoso.com

Get Started

1

Clone the repo

git clone https://github.com/prathiraj/teams-quick-chat.git
cd teams-quick-chat
2

Build & publish

dotnet publish -c Release

Output: bin\Release\net9.0-windows\win-x64\publish\TeamsQuickChat.exe

3

Run & pin

Run the exe. Find the chat-bubble icon in your system tray. Drag it onto the taskbar to pin it.

4

Auto-start (optional)

Press Win+R → shell:startup → drop a shortcut to the exe there.
